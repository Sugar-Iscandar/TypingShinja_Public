# TypingShinja
プログラミングの練習として、Unityで作っているタイピングゲームです。

大きく分けて「初級、中級、上級、特級」という難易度が存在し、各難易度ごとに5~6のステージが存在します。

本ゲームの動作にはUniRxの導入が必要です。
